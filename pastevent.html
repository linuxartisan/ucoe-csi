Workshop for Python Programming

On 5th March, 2019 about 20 students attended a hands-on workshop for Python Programming, conducted at Universal College of Engineering under the CSI chapter. The workshop was conducted by Mr. Hrishikesh Gadkari and Mr. Jeet Patel.
Firstly the students were explained the 3 different ways to run a Python program. The best way was using Spyder IDE which comes with Anaconda Python. Basic syntax of Python, operators in Python, loops, and simple Python programs were the major topics covered during the session which gave the students a brief idea about Python.

Seminar on Responsive Website Designing

CSI student's chapter of Universal College of Engineering organized Responsive Website Designing seminar which was conducted on 20th March, 2019. About 15 students attended this seminar. This seminar was conducted by Mr. Ashutosh Mishra and Mr. Archit Nangalia. Responsive web design (RWD) is an approach to web design that makes web pages render well on a variety of devices and window or screen sizes. Recent work also considers the viewer proximity as part of the viewing context as an extension for RWD. Content, design and performance are necessary across all devices to ensure usability and satisfaction.
